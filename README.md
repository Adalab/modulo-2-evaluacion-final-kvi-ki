# Evaluación final (JavaScript module) de Victoria Kovaleva.

Es una página para buscar las películas o las series desde la web https://www.tvmaze.com/. Puedes crear una lista de tus películas/series favoritas, solo tienes picar a la imagen o al nombre de ítems.

En este proyecto he usado [API](http://api.tvmaze.com/search/shows?q=:query). Los lenguajes son HTML, CSS y JavaScript.

He utilizado bucles, addEventListener, fetch, DOM y localStorage.
